# Slot-Based-Monthly-Attendance-Calculator
In this Mini Project, it is Calculator Developed for **Student's Personal Use** to **Calculate their Monthly Attendance Based on Slots** According to their College's timing For Example -> **Morning Slot: (10AM - 12PM), Afternoon Slot: (1-3PM) & Evening Slot: (3-5PM)** . Students must have to Select the Type of Month First for the Final Output.
