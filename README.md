# Slot-Based-Monthly-Attendance-Calculator
In this Mini Project, it is Calculator Developed for **Student's Personal Use** to **Calculate their Monthly Attendance Based on Slots** According to their College's timing For Example -> **Morning Slot: (10AM - 12PM), Afternoon Slot: (1-3PM) & Evening Slot: (3-5PM)** . Students must have to Select the Type of Month First for the Final Output.

# WorkFlow:
1. First, Select the Month's Type Option -> (28Days/30Days/31Days/15Days)
2. Student must Input the **Total Morning, Afternoon, Evening Slots S/He have Attended throughout the whole Month** based on the **Number of Days of Month(Sundays are Eliminated)**.
3. If S/He have Entered a greater value > Than the number number of Days of Month(Sundays are Eliminated) "INVALID NUMBER" will be Displayed.
4. After giving all Inputs, the Total Number of Slots -> (Morning, Afternoon, Evening) Attended of the Selected Month will be Showed.
5. At the Last, the Percentage of Attendance will be Displayed at the Screen.
